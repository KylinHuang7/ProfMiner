# 🤍 蛋白石系列

蛋白石是一种类似石英的华丽装饰材料，外观呈乳白色并带有优雅花纹。

## 蛋白石矿石（Opal Ore）

![蛋白石矿石](../screenshots/opal/opal_ore.png)

- **生成范围**：Y = -40 ~ 40，均匀分布
- **矿脉大小**：每个矿脉最多 32 个方块，每区块 5 次
- **挖掘要求**：石镐及以上
- **掉落**：3~6 个蛋白石（支持时运附魔，精准采集掉落矿石本身）
- **不掉落经验**

## 蛋白石（Opal）

![蛋白石](../screenshots/opal/opal.png)

- 蛋白石矿石的掉落物，用于合成蛋白石块
