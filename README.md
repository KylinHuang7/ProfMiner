# Professional Miner

一个专注于各种矿物的模组。
支持MineCraft 1.21.1

---

## 📦 新增物品一览

### 💎 [红宝石系列](docs/ruby.md)

![红宝石矿石](screenshots/ruby/ruby_ore.png)

稀有矿物，生成于深层地下（Y = -59 ~ -10）。开采红宝石矿石获得红宝石，6 个红宝石合成红宝石心，使用后永久增加 1 颗红心上限。

> 包含物品：红宝石矿石、深板岩红宝石矿石、红宝石、红宝石心
>
> 👉 [查看详细信息](docs/ruby.md)

---

### 🤍 [蛋白石系列](docs/opal.md)

![蛋白石全家福](screenshots/opal/opal_suite.png)

类似石英的华丽装饰材料，外观呈乳白色并带有优雅花纹。生成于中层地下（Y = -40 ~ 40），矿脉丰富，可加工为多种建筑方块。

> 包含物品：蛋白石矿石、蛋白石、蛋白石块、平滑蛋白石块、蛋白石砖、蛋白石楼梯、蛋白石台阶、蛋白石压力板、蛋白石围墙
>
> 👉 [查看详细信息](docs/opal.md)

---

## 🎨 创造模式

所有物品可在以下创造模式标签页中找到：
- **建筑方块**：蛋白石矿石、蛋白石块、平滑蛋白石块、蛋白石砖、蛋白石楼梯、蛋白石台阶、蛋白石压力板、蛋白石围墙、红宝石矿石、深板岩红宝石矿石
- **材料**：红宝石、红宝石心、蛋白石
- **Professional Miner 标签页**：包含全部物品
